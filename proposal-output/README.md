# 项目书材料包

本目录收纳投稿版与公开展演版所需的全部材料。

## 文件说明

- `01_重点课题版.md`：项目书主文稿，建议作为投稿主文本
- `02_短版.md`：短摘要版，适合填报或快速审阅
- `03_封面目录导师意见模板.md`：封面、目录与签批模板
- `04_深度调研报告.md`：文献调研与相关工作支撑
- `05_公开展演提纲.md`：公开展演、答辩与演示提纲

## 配套材料

- `images/cover.png`：封面插图
- `images/evaluation.png`：评价图
- `images/roadmap.png`：路线图
- `../docs/ml-benchmark-presentation.html`：PPTX 风格 HTML 演示页
- `../docs/ml-benchmark-report.md`：基准实验报告
- `../reports/ml-sweep-20260506-160249/`：实验结果与图表

## 推荐提交流程

1. 先通读 `01_重点课题版.md`，补齐个人、团队、签字类信息。
2. 用 `03_封面目录导师意见模板.md` 生成最终封面与签批页。
3. 将 `04_深度调研报告.md` 作为附件或支撑材料。
4. 公开展演时配合 `05_公开展演提纲.md` 与 `docs/ml-benchmark-presentation.html` 使用。
